## Usage

### Run the server

* `node start`

### Client

* open a web browser to `localhost:3000`
* Select a name and carbonated beverage flavor
* Click Join
* Open additional tabs/windows to add more players
* Click Start Game on any client

## Controls

Arrow Keys:

Left/Right: move
Up: jetpack thrust
